POS All In One Features

This module is used for reorder,return order,return order with barcode,reprint invoice, filter order based on customers, apply gift coupon & vouchers, apply bag charges, get invoice at a time of payment,get all the records in POS, POS Fixed Amount Discount & Display Total Number of Purchased Items in Cart and POS Receipt.

key Features:

  • All Users orders
  • Re-Order
  • Return Order
  • Return Order with Barcode
  • Re-Print Invoice
  • Apply Gift Coupon & Vouchers
  • Add Bag Charges
  • Create Sale Order
  • Import Sale Order
  • Generate Invoice automatically at time of payment
  • Filter Order based on customer
  • Show on hand quantity of product
  • Display Total Number of Purchased Items in Cart and POS Receipt (POS Items Count).
  • Add Fixed Amount along with Percentage Discount according to configuration.

How to use

Configuration

Select from options to set limit to load pos orders


All User Order

We can see all orders by clicking "Orders" Button .


Re-Order

We can re-order same product by clicking "Re-Order" Button.


Return Product

Return Order button is used to return products from POS created order.



After clicking on Return Order button, Return product popup is open, In that popup you can fill products quantities which you want to return from POS return order.

Return Order




Return order With Barcode

In POS Screen "Return Order with Barcode" button appear, Click to Scan a Barcode for a particular order.




Scan the Barcode

Scan the Barcode and Barcode will automatically read in the input field in the Popup. Then Click Apply.




Order for the scanned barcode will be read and Orderlines will be added in the popup window




After clicking on Apply, Products will be added to POS Order with Negetive Quantity and you can place the Return Order.




Return order receipt

After return order payment done, see payment receipt.




Re-Print Invoice

We can Re-Print invoice with "Reprint" button.


Create Sale Order

We can re-order same product by clicking "Re-Order" Button.


Import Sale Order

We can import sale-order by clicking "Sale-Order" Button.


When you click on sale orders button it will redirects you to list of sale orders.


After clicking on Import Order button, Sale order's products popup is open, In that popup you can fill products quantities which you want to import from sale order.


Imported-Order


Sale order which is imported in POS.


If product is not configured for pos(available in pos),then it will generate a popup.


Configuration

Check the field to import sale order in POS.


Make sure that available in POS field is checked under the Product.


Create Invoice at time of payment

Activating Invoice auto-check button, it automatically creates Invoice after payment is done.


Filter Orders

We can filter orders by customers name.


POS Stock

The available product quantity is display in the POS.


Stock Location Configuration

In POS you can display product stock as per stock locations.


If you select show stock location as "Specific" stock of specific stock location is dislay on POS.


Display stock of product by their stock location.


Display stock of product of speicfic stock location that select at time of configuration in POS.


Stock Quantity Validation with "All Warehouse" Configuration. If You add quantity more than available quantity in Orderline for a particular product then Stock Qty Validation will be added.


Stock Quantity Validation with "Current Session Warehouse" Configuration. If You add quantity more than available quantity in Current Warehouse in Orderline for a particular product then Stock Qty Validation will be added.


POS Coupons & Gift Vouchers

Gift Voucher and Coupon code can be apply in POS Orders. Print Gift Coupon option available.


Add Bag Charges

This module allows us to add carry bag charges on particular order.


POS Items Count

Display Total Number of Purchased Items in Cart and POS Receipt.


POS Fixed Amount Discount along with Percentage

This Feature allow your customer to fixed amount discount option along with percentage.
