Odoo 12 POS Cross Selling


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Manage Cross Selling Items.

How to use

  • Click on "Configuration >> Product Cross-Selling".
  • Configure cross selling products with price after applying discount.


  • Select Cross selling Products.


  • It is keeping history of cross selling products.
  • To view history click on "Configuration >> Cross-Selling History".
